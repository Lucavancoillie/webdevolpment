Waarom staat de tekst in de blokken horizontaal gecentreerd?

Omdat bij de div van de css staat text-align center.

Ga na dat een property vertical-align:center geen effect heeft. Waarom eigenlijk niet?

Het vertical-align: center; heeft geen direct effect in CSS, omdat er geen "center" waarde is voor vertical-align.

baseline: Het element wordt uitgelijnd op de basislijn van de omliggende tekst. top: Het element wordt uitgelijnd op de bovenste rand van de lijn. bottom: Het element wordt uitgelijnd op de onderste rand van de lijn. middle: Het element wordt uitgelijnd in het midden van de lijn. zijn 4 waarden die je kan gebruiken.

Ga na dat vertical-align:middle er niet voor zorgt dat de tekst verticaal gecentreerd wordt. Wat doet vertical-align dan wel?

het zet alle blokjes mooi gencenterd naast elkaar staan dus mooi allemaal in het midden staan en de groter blokjes dan gewoon meer uitsteken boven en beneden van een denkbeeldige lijn.

Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren? (tip:line-height)

De 'simpele' manier om een enkele regel tekst verticaal te centreren is door gebruik te maken van de line-height eigenschap. Je kunt de line-height gelijk maken aan de hoogte van het omliggende element. Dit zal de tekst verticaal centreren binnen dat element.

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren? (tip: table cell)

Een veelgebruikte truc om meerdere regels tekst verticaal te centreren is het gebruik van tabelcellen (display: table-cell). Door een container in te stellen als een tabel en de inhoud ervan als een tabelcel, kun je de tekst verticaal centreren, zelfs als deze uit meerdere regels bestaat.

In dit voorbeeld wordt de tekst binnen een element met de class .content verticaal gecentreerd, zelfs als het meerdere regels bevat. Door de container in te stellen als een tabel en de inhoud als een tabelcel, wordt de verticale centrering bereikt met behulp van vertical-align: middle.

In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang?

In CSS krijgt de regel die later in het stylesheet wordt gedefinieerd voorrang als er conflicterende stijlregels zijn voor hetzelfde element.
